iPrintOS 自助列印 — 常見問題

無人自助門市 · 列印 / 影印 / 掃描 / 付款 一次搞懂

本店為 無人自助門市，現場沒有專人。多數狀況可自行排除；機台問題時，店內共有三台機器（每台間隔約一公尺），可直接改用旁邊另一台，未完成的列印不會扣款。

📱 手機 / 雲端檔案列印

我要印手機裡的檔案
請拿手機掃描機台上的 QR Code，就能上傳檔案列印。
QR Code 在哪裡？
在機台螢幕或機身貼紙上，掃描後手機會打開上傳頁面。
支援什麼檔案格式？
支援 PDF、Word，以及 JPG、PNG 圖片，建議先轉成 PDF 最穩定。
LINE 裡的檔案怎麼印？
先在 LINE 把檔案存到手機，再掃 QR Code 從上傳頁選那個檔案。
我要印照片
掃 QR Code 後選照片上傳，可以選彩色和尺寸再確認列印。
怎麼選黑白或彩色？
上傳後在手機頁面可切換黑白或彩色，彩色費用較高。
怎麼設定份數？
在手機預覽頁面可調整份數，確認後再付款。
可以雙面列印嗎？
可以，在手機設定頁面選「雙面列印」即可。
紙張大小怎麼選？
可選 A4 或 A3，在手機頁面的紙張設定切換。
列印前可以先預覽嗎？
可以，付款前手機會顯示預覽，確認沒問題再付款。
檔案太大傳不上去
檔案較大請稍等上傳完成，或先壓縮、分次上傳。
上傳後找不到檔案
請確認有「上傳完成」提示，沒有的話重新掃 QR Code 再上傳一次。
上傳完怎麼付款？
確認預覽後點付款，依畫面用電子支付或掃碼完成即可。
印好的紙在哪裡拿？
付款完成後紙張會從機台出紙匣送出，請到出紙口拿取。
可以一次印很多檔案嗎？
可以，上傳頁面能加入多個檔案，會一起計算份數與價格。
Word 排版跑掉了
Word 轉檔有時會位移，建議先在手機轉成 PDF 再上傳最準確。
手機沒網路怎麼辦？
請開啟手機行動網路，或連線店內提供的網路後再掃 QR Code。
上傳到一半想取消
直接關閉手機頁面即可取消，未付款不會列印也不會收費。
可以縮放或滿版嗎？
在手機設定可選縮放或滿版列印，預覽會即時更新。
列印怎麼算錢？
系統會依頁數、黑白或彩色、紙張大小自動計價，付款前會顯示金額。
印一份要等多久？
付款後通常幾秒到十幾秒就會出紙，份數多會久一點。
我的檔案會被留下來嗎？
您的檔案只暫存在店內主機處理，列印後會自動清除，不會上傳雲端。

🪪 身分證 / 證件影印

我要影印身分證
請把證件放在影印機玻璃面板上，蓋上蓋子，依畫面操作影印。
正反面要印在同一張
可選「證件影印」模式，會把正反面合印在同一張紙上。
證件要放哪裡？
放在玻璃面板左上角對齊標示線，印好再翻面印背面。
健保卡 / 駕照可以印嗎？
可以，健保卡、駕照都用同樣的證件影印方式操作。
護照怎麼印？
護照翻到資料頁，平放在玻璃面板上影印即可。
可以放大嗎？
可以，在畫面選放大或縮小比例後再影印。
證件要黑白還是彩色？
可選黑白或彩色，彩色費用較高，依畫面切換。
我要印好幾份
在畫面設定份數後再確認影印。
影印證件多少錢？
系統會依黑白彩色與份數計價，操作畫面會顯示金額。
影印也要先付款嗎？
是的，系統會先預覽報價，付款後才會印出來。
印出來歪歪的
請把證件對齊玻璃面板角落的標示線，再重印一次。
我有好幾張證件
可以一張一張放上去分別影印，或用證件影印模式逐張處理。
印好的在哪拿？
付款完成後請到機台出紙口拿取。
證件影印要多久？
通常幾秒就會出紙。

🖨️ 掃描到 Email / USB

我要掃描文件
請把文件放上玻璃面板或送稿器，在畫面選掃描方式。
我要掃描到信箱
選「掃描到 Email」，輸入您的信箱，掃描後檔案會寄過去。
我要掃到 USB
選「掃描到 USB」，把隨身碟插進機台插槽再開始掃描。
USB 插哪裡？
USB 插槽在機台前方面板上，插好畫面會顯示已偵測。
Email 沒收到
請確認信箱輸入正確並查看垃圾信件匣，必要時重新掃描寄送。
可以掃很多頁嗎？
可以，用上方送稿器一次放多張，會合成一個檔案。
掃彩色還是黑白？
可在畫面選彩色或黑白掃描。
掃成 PDF 還是圖片？
可選 PDF 或 JPG，多頁文件建議選 PDF。
原稿放哪？
單張放玻璃面板，多張放上方送稿器。
掃描要錢嗎？
掃描費用依機台設定，操作畫面會顯示是否收費與金額。
可以調解析度嗎？
可以，在畫面選解析度，越高檔案越大。
可以雙面掃嗎？
可以，用送稿器並在畫面選雙面掃描。
檔案太大寄不出去
請降低解析度或分次掃描，檔案會比較小。
掃描的檔案會留著嗎？
掃描檔只暫存於店內主機，完成寄送或存檔後即清除，不上雲。

💳 付款

怎麼付款？
確認預覽後點付款，依畫面用電子支付或掃碼完成。
可以用什麼付款？
機台支援掃碼與電子支付，實際方式以付款畫面顯示為準。
可以用 LINE Pay 嗎？
若付款畫面有顯示該選項即可使用，依畫面掃碼付款。
可以刷悠遊卡嗎？
是否支援以付款畫面顯示為準，請依畫面選擇方式。
可以刷卡嗎？
信用卡是否支援以付款畫面為準，依畫面操作即可。
可以投現金嗎？
是否收現金以機台設定為準，畫面會顯示可用的付款方式。
付款了卻沒出紙
請先確認出紙匣並稍等約 30 秒；若仍沒出紙，請直接改用旁邊另一台機器重印，未完成的列印不會扣款。
我好像被扣兩次
不好意思造成困擾；目前現場無法直接處理重複扣款，請先保留您的交易紀錄，線上退款申請功能即將開放，屆時可提出查證。
付款失敗
請稍候再試一次，或換一種付款方式；也可以直接改用旁邊另一台機器。
付款的 QR 在哪？
確認預覽後，付款 QR 會顯示在手機或機台畫面上。
有收據或發票嗎？
電子發票或收據會依畫面顯示，您可以在手機操作頁面查看。
金額好像不對
金額會依頁數、彩色與紙張大小自動計算，付款前畫面都會先顯示金額；若仍有疑問，請先保留交易紀錄，線上查證功能即將開放。
付到一半中斷了
未完成付款不會列印也不會扣款，請重新操作一次。
我要退款
不好意思，若是付款後沒印出來，您可以直接改用旁邊另一台機器重印，未完成的列印不會扣款；線上退款申請功能即將開放。
有找零或低消嗎？
相關規則以畫面與機台設定為準，操作前畫面會顯示金額。
付款後多久會印？
付款成功後通常幾秒內就會開始出紙。

🛠️ 機台狀況（卡紙 / 缺紙 / 故障）

卡紙了
請依機台螢幕的圖示指示打開紙匣、取出卡住的紙張即可；若不想自行處理，旁邊還有另外兩台機器可以直接使用，未完成的列印不會扣款。
沒紙了
請依機台螢幕指示打開紙匣補充紙張；或直接改用旁邊另一台機器，未完成的列印不會扣款。
印出來是空白的
通常是檔案或設定問題，請檢查原始檔案後重新列印，或改用旁邊另一台機器試試。
印出來有條紋很髒
列印品質不佳時，建議直接改用旁邊另一台機器列印。
機器沒反應 / 當機
請直接改用旁邊另一台機器操作，未完成的訂單不會扣款。
螢幕黑掉了
這台螢幕沒反應時，請改用旁邊另一台機器。
印到一半停住
請稍候幾秒；若沒有恢復，可改用旁邊另一台機器重印。
顏色很淡 / 沒碳粉
列印偏淡時，建議改用旁邊另一台機器列印。
我可以自己把紙拉出來嗎？
可以，請依機台螢幕的圖示指示慢慢取出卡住的紙張，不要硬拉；不確定的話直接改用旁邊另一台機器。
紙要怎麼補？
請依機台螢幕指示打開紙匣，放入紙張即可。
我印錯了
已印出的內容無法收回；如需重印請重新操作，或改用旁邊另一台機器。
機器有怪聲音
機台有異常聲響時，請停止使用這台，改用旁邊另一台機器。
蓋子 / 送稿器卡住
請依機台螢幕指示輕輕復位，不要強行扳動；不確定就改用旁邊另一台機器。
機器壞掉了
這台無法使用時，請直接改用旁邊另一台機器，未完成的訂單不會扣款。
