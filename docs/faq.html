iPrintOS 自助列印 — 常見問題

無人自助門市 · 列印 / 影印 / 掃描 / 付款 一次搞懂

本店為 無人自助門市，現場沒有專人。多數狀況可自行排除；機台問題時，店內共有三台機器（每台間隔約一公尺），可直接改用旁邊另一台，未完成的列印不會扣款。
門市資訊：24 小時營業、全年無休｜無停車位｜店內掃碼網路僅供傳檔、不提供上網｜不提供護貝/裁切/相片紙/特殊磅數｜不提供發票（付款明細顯示於畫面）

📱 手機 / 雲端檔案列印

我要印手機裡的檔案
好的，請拿手機掃描機台上的 QR Code，就能上傳檔案列印。
QR Code 在哪裡
QR Code 在機台螢幕或機身貼紙上，掃描後手機會打開上傳頁面。
支援什麼檔案格式
支援 PDF、Word、以及 JPG、PNG 圖片，建議先轉成 PDF 最穩定。
LINE 的檔案怎麼印
先在 LINE 把檔案存到手機，再掃 QR Code 從上傳頁選那個檔案。
我要印照片
掃 QR Code 後選照片上傳，可以選彩色和尺寸再確認列印。
要選黑白還是彩色
上傳後在手機頁面可切換黑白或彩色，彩色費用較高。
怎麼設定份數
在手機預覽頁面可調整份數，確認後再付款。
可以雙面印嗎
可以，在手機設定頁面選「雙面列印」即可。
紙張大小怎麼選
可選 A4 或 A3，在手機頁面的紙張設定切換。
印之前可以先看嗎
可以，付款前手機會顯示預覽，確認沒問題再付款。
檔案太大傳不上去
檔案較大請稍等上傳完成，或先壓縮、分次上傳。
上傳後找不到檔案
請確認上傳完成的提示，沒有的話重新掃 QR Code 再上傳一次。
上傳完怎麼付款
確認預覽後點付款，依畫面用電子支付或掃碼完成即可。
印好的紙在哪裡拿
付款完成後紙張會從出紙匣送出，請到出紙口拿取。
可以一次印很多檔案嗎
可以，上傳頁面能加入多個檔案，會一起計算份數與價格。
Word 排版跑掉了
Word 轉檔有時會位移，建議先轉成 PDF 再上傳最準確。
手機沒網路怎麼辦
掃機台 QR Code 上傳檔案時會自動連線傳檔，不需另外上網；其他用途請用手機行動網路。
上傳到一半想取消
直接關閉手機頁面即可取消，未付款不會列印也不會收費。
可以縮放或滿版嗎
在手機設定可選縮放或滿版列印，預覽會即時更新。
列印怎麼算錢
系統會依頁數、黑白或彩色、紙張大小自動計價，付款前會顯示金額。
印一份要等多久
付款後通常幾秒到十幾秒就會出紙，份數多會久一點。
我的檔案會被留下來嗎
您的檔案只暫存在店內主機處理，列印後會自動清除，不會上傳雲端。

可以，平板一樣掃 QR Code 上傳即可。
電腦檔案可以印嗎
可以，把檔案傳到手機或用手機開雲端，再掃 QR Code 上傳。
雲端硬碟的檔案怎麼印
先在手機開啟雲端把檔案下載，再掃 QR Code 上傳。
Email 附件怎麼印
在手機把附件存下來，再掃 QR Code 從上傳頁選它。
印簡報 / PPT

印 Excel 表格
可以，建議先轉 PDF 並確認列印範圍再上傳。
