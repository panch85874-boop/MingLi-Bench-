## 🔧 拿去生成口播視頻的做法（擇一）

- **剪映 / CapCut**：新建直式專案 → 文字朗讀（AI 配音，選活潑女聲/男聲）貼上「口播稿」→ 按分鏡放素材 → 自動字幕。零成本、最快。
- **HeyGen / D-ID / Synthesia**：要「數字人真人口播」就用這類。貼「口播稿」→ 選 avatar + 中文語音 → 匯出 9:16。
- **可靈 / Pika / Runway**：要動態 B-roll（猩猩、飛鏢、輪盤特效）可生成補鏡頭，再回剪映合成。

> 建議語音：台灣口音中文、語速中等偏快；BGM 用輕快俏皮的 royalty-free。
