# 🐵 台股千金俄羅斯輪盤 · 黑猩猩射飛鏢

一款純前端、零相依的台股「千金股」隨機選股遊戲模擬器。

靈感來自華爾街經典故事——**讓矇眼的黑猩猩對著股票版面射飛鏢來選股**
（出自 Burton Malkiel《漫步華爾街》），用輪盤把「隨機選股」變成有驚喜、有遊戲感的互動體驗。

## ✨ 特色

- 🎡 **Canvas 彩色輪盤**：38 檔台股千金/準千金股，色塊取材自輪盤圖。
- 🎯 **飛鏢指針 + 猩猩按鈕**：點擊中央 🐵 即可「射飛鏢」轉動輪盤。
- 🔊 **純 Web Audio 音效**（免外部檔案）：
  - 飛鏢射出「咻」聲
  - 輪盤經過卡榫的「噠噠」聲
  - 命中時的勝利號角聲
- 🎉 **落點驚喜**：命中後彩帶噴發。
- 🔍 **放大顯示 5 秒**：單獨放大顯示「股票名稱 + 代號 + 建議購買 1~10 股」，
  右上角有 5 秒倒數（亦可點背景提早關閉）。
- 🧾 **歷史落點**：側欄記錄最近 12 次結果。

## ▶️ 執行方式

直接用瀏覽器開啟 `index.html` 即可，無需安裝任何套件：

```bash
# 任一方式
open game/index.html              # macOS
xdg-open game/index.html          # Linux
# 或啟動簡易伺服器
python3 -m http.server -d game 8000   # 開 http://localhost:8000
```

## 📁 檔案結構

| 檔案 | 說明 |
|---|---|
| `index.html` | 遊戲主畫面與版面 |
| `styles.css` | 樣式、動畫、放大顯示與彩帶效果 |
| `game.js` | 輪盤繪製、轉動物理、音效、命中判定與 5 秒放大顯示 |
| `stocks.js` | 千金股清單（代號／名稱／色塊） |
