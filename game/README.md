# 🐵 台股千金俄羅斯輪盤 · 黑猩猩射飛鏢

一款**單一檔案、零相依**的台股「千金股」隨機選股遊戲模擬器。整個遊戲(股票清單、樣式、程式、音效)全部內建在 `index.html` 一個檔案裡。

靈感來自華爾街經典故事——**讓矇眼的黑猩猩對著股票版面射飛鏢來選股**
（出自 Burton Malkiel《漫步華爾街》），把「隨機選股」變成有驚喜、有遊戲感的互動體驗。

## ✨ 特色

- 📖 **開場故事頁**：打開先看「黑猩猩射飛鏢」的故事與啟示，點擊確認後才進入遊戲。
- 🎡 **Canvas 彩色輪盤**：38 檔台股千金/準千金股。
- 🎯 **飛鏢指針 + 猩猩按鈕**：點中央 🐵 即可「射飛鏢」轉動輪盤。
- 🔊 **純 Web Audio 音效**（免外部檔案）：射出「咻」聲、輪盤「噠噠」聲、命中號角聲。
- 🎉 **落點驚喜**：命中後彩帶噴發。
- 🔍 **放大顯示 5 秒**：放大顯示「股名 + 代號 + 建議購買 2~6 股」（千金股單價高，股數偏少），含 5 秒倒數。
- 📤 **一鍵分享**：手機用系統分享、電腦自動複製連結，方便貼 FB / LINE。
- 📱 **手機友善**：可「加到主畫面」像 App 一樣使用。

## ▶️ 使用方式

**單檔遊玩**：直接用瀏覽器打開 `index.html` 即可。

**部署成公開網址（手機可玩、可貼 FB）**：用 GitHub Pages 最簡單。
啟用後網址為：`https://<帳號>.github.io/<倉庫名>/game/`

```bash
# 本機預覽
python3 -m http.server -d game 8000   # 開 http://localhost:8000
```
